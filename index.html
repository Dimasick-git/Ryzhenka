Инструкция по установке

  1. Скачайте архив из раздела «Релизы» (ветка: Ryazhenka или AIO/Lite).
  2. Распакуйте на карту памяти microSD (exFAT/FAT32). Сохраните бэкапы.
  3. Запустите payload через ваш загрузчик (hekate/RCM), следуйте шагам установщика.
  4. Выберите нужный профиль (OC/сток), при первом запуске выполните настройку Tuner v8.0.
  5. Для обновлений заменяйте файлы из нового релиза, читайте CHANGELOG.
