  ОПИСАНИЕ / DESCRIPTION
═══════════════════════════════════════════════════════════════════

Добро пожаловать в DemoTools - демонстрационный пакет кастомного
функционала для прошивки Nintendo Switch!

Этот пакет создан для демонстрации возможностей кастомизации
и расширения функционала домашней прошивки в стиле тюнерных тулов.

  СОДЕРЖИМОЕ ПАКЕТА / PACKAGE CONTENTS
═══════════════════════════════════════════════════════════════════

📦 DemoTools/
   ├── package.ini           - Основной конфигурационный файл
   ├── demo_readme.txt       - Этот файл с описанием
   └── Data/                 - Папка с дополнительными данными

═══════════════════════════════════════════════════════════════════
  УСТАНОВКА / INSTALLATION
═══════════════════════════════════════════════════════════════════

1. Скопируйте папку DemoTools в:
   /switch/.packages/DemoTools/

2. Убедитесь, что все файлы на месте:
   ✓ package.ini
   ✓ demo_readme.txt
   ✓ Data/ (папка)

3. Перезагрузите консоль

4. Найдите DemoTools в меню пакетов

═══════════════════════════════════════════════════════════════════
  ИСПОЛЬЗОВАНИЕ / USAGE
═══════════════════════════════════════════════════════════════════

1. Откройте меню пакетов в вашей прошивке
2. Выберите раздел DemoTools
3. Изучите доступные опции
4. Тестируйте функционал в безопасной среде
5. Адаптируйте под свои нужды

═══════════════════════════════════════════════════════════════════
  СТРУКТУРА КОНФИГУРАЦИИ / CONFIG STRUCTURE
═══════════════════════════════════════════════════════════════════

package.ini использует формат конфигурации для homebrew меню:

[Секция]        - Заголовок секции
;комментарий    - Комментарий (начинается с ;)
'Опция' = ''    - Пункт меню
команда         - Исполняемая команда
